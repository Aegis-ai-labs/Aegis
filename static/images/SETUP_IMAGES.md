# Hardware Images Setup

This directory contains images for the AEGIS1 project.

## Required Images

### 1. Real Hardware Photo
Place a photo of the actual ESP32 pendant hardware here:
- **Filename:** `aegis-hardware.jpg` or `aegis-hardware.png`
- **Description:** Photo of the assembled ESP32 pendant with mic and speaker

### 2. Prototype Design
Download the prototype design from:
https://drive.google.com/file/d/1V1G8TYWNelViNsZtqQEyoUdi4ldkK9Ss/view?usp=drive_link

- **Filename:** `prototype-design.png`
- **Description:** Prototype design mockup/concept image
- **Instructions:** 
  1. Open the Google Drive link above
  2. Download the image
  3. Save it as `prototype-design.png` in this directory

## Usage

These images are referenced in:
- `README.md` - Main project documentation
- `static/index.html` - Web dashboard
- `firmware/README.md` - Hardware documentation
